Kebijakan Privasi Cermis Audio

Terakhir diperbarui: Juni 2026

Selamat datang di Cermis Audio. Kami sangat menghargai privasi Anda sebagai pengguna aplikasi kami. Kebijakan Privasi ini menjelaskan bagaimana aplikasi kami bekerja dan menegaskan komitmen kami bahwa kami tidak mengumpulkan data pribadi Anda.

Inti Kebijakan Privasi Kami: Aplikasi Cermis Audio dirancang agar dapat digunakan tanpa login, tidak mengumpulkan data pribadi di server, dan sepenuhnya memanfaatkan penyimpanan lokal di perangkat Anda.

1. Pengumpulan dan Penggunaan Data

Kami tidak mengumpulkan, tidak menyimpan, dan tidak membagikan informasi pribadi apa pun seperti nama, alamat email, nomor telepon, atau lokasi geografis sensitif Anda. Aplikasi ini dapat digunakan langsung tanpa perlu membuat akun atau melakukan login.

2. Penyimpanan Lokal (SharedPreferences)

Untuk memberikan pengalaman pengguna yang baik, aplikasi kami menggunakan fitur penyimpanan lokal pada sistem operasi Android Anda (dikenal sebagai SharedPreferences). Data yang disimpan di perangkat Anda meliputi:

  • Riwayat Membaca & Mendengarkan: Catatan cerita mana saja yang sudah Anda buka, semata-mata agar Anda dapat melacak riwayat bacaan Anda sendiri.
  • Data Akses Waktu (Reward): Durasi sisa waktu akses setelah Anda menonton iklan reward.

Data ini tetap berada di dalam perangkat Anda dan tidak pernah dikirim ke server kami ataupun pihak ketiga. Anda dapat menghapus data ini kapan saja dengan menghapus penyimpanan aplikasi melalui pengaturan sistem Android Anda.

3. Sistem Akses dan Iklan Pihak Ketiga (Reward Ads)

Cermis Audio menggunakan layanan iklan pihak ketiga untuk membantu mendanai operasional aplikasi. Aplikasi ini menyediakan sistem Akses Reward 1 Jam, di mana Anda dapat menonton video iklan reward untuk membuka akses penuh ke seluruh konten cerita selama 60 menit.

Penyedia layanan iklan kami dapat mengumpulkan data perangkat anonim untuk keperluan personalisasi iklan, seperti:

  • ID Iklan Google (AAID).
  • Informasi teknis perangkat (model perangkat, versi OS).

Layanan pihak ketiga yang kami gunakan meliputi:

  • Google AdMob: Kebijakan Privasi Google

4. Keamanan

Karena kami tidak meminta atau menyimpan data pribadi Anda di server eksternal, risiko kebocoran data di sisi server adalah nihil. Keamanan riwayat membaca Anda sepenuhnya terlindungi di dalam ruang penyimpanan aman internal aplikasi di perangkat Android Anda.

5. Perubahan Kebijakan Privasi

Kami dapat memperbarui Kebijakan Privasi ini dari waktu ke waktu untuk menyesuaikan dengan pengembangan fitur aplikasi kami. Perubahan apa pun akan langsung dipublikasikan di halaman web ini.

6. Hubungi Kami

Jika Anda memiliki pertanyaan mengenai implementasi privasi pada aplikasi Cermis Audio, Anda dapat menghubungi kami melalui halaman repositori resmi kami di GitHub.
